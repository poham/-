# Camera Gizmo 透視角度更新

## 改進說明

將 3D Gizmo 的視角從「正視圖」調整為「3/4 透視圖」，讓使用者更容易理解球體的立體空間。

## 視角調整

### 之前（正視圖）：
- 方位角 0° 時，攝影機在正中央
- 球體看起來是平面的
- 難以理解 3D 空間關係

### 現在（3/4 透視圖）：
- 整個球體向左旋轉 45°
- 方位角 0° 時，攝影機在右前方（從觀察者角度）
- 可以清楚看到球體的立體感
- 更符合 3D 軟體的視角習慣

## 技術實作

```typescript
const viewRotation = -45; // 向左旋轉 45 度

// 在計算攝影機位置時加入視角旋轉
const azimuthRad = ((azimuth + viewRotation) * Math.PI) / 180;

// 圓圈也需要相應旋轉
transform={`rotate(${viewRotation} ${center} ${center})`}
```

## 視覺效果

- **紅色圈（Azimuth）**：呈現橢圓形，顯示水平旋轉軌跡
- **藍色圈（Elevation）**：前方可見部分，顯示垂直角度
- **綠色圈（輔助）**：側面參考線
- **標籤位置**：調整到更符合透視角度的位置

## 使用者體驗提升

✅ 更直觀的 3D 空間理解
✅ 符合專業 3D 軟體的視角習慣
✅ 清楚看到攝影機在球面上的移動軌跡
✅ 0° 位置更有意義（正面視角）
